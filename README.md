# MNNIT Buzz

A platform for MNNITians.

## Features

- [x] Share your posts. Write anything.
- [x] Upvotes and downvotes to make useful posts popular.
- [x] Comment on posts.
- [x] Follow MNNITians.
- [x] Search for MNNITians.
